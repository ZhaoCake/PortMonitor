## Quick Start

安装UV。

```bash
uv sync
```

激活虚拟环境。

```bash
source .venv/bin/activate   # MacOS/Linux
. .venv/Scripts/activate    # Windows
```

运行主程序

```bash
python ./WindowManager.py
```
