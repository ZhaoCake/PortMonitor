# PortMonitor

一个运用PYQT5的串口监视工具。

[详细信息](./docs/docs.md)

---

## Quick Start

安装UV。

```bash
uv sync
```

激活虚拟环境。

```bash
source .venv/bin/activate   # MacOS/Linux
. .venv/Scripts/activate    # Windows
```

运行主程序

```bash
python ./WindowManager.py
```
